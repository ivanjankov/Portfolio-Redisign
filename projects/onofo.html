Onofo

Onofo is a SaaS company. Their main product is AI-Powered notes taking platform and app.
This is not a website for a client*

Technologies Used: HTML, CSS, JavaScript

Product Description

Onofo is used to take notes while in a meeting, class or learning online. They address individuals (you can buy a personal subscription), and companies/schools (you can buy subscriptions for a company/school). Onofo it’s cross-platform, which means you can either use it within a browser or mobile and tablet. Some of the features their platform offer are: Real-time structuring of notes, Group collaboration, Scoring.
